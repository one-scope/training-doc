# コマンド置換

コマンド置換は、コマンドの実行結果を他のコマンドや変数に渡すための機能です。  
コマンド置換は ``` `` ``` または `$()` を使って表現します。  
どちらも同じ意味ですが、後者の `$()` の方がネストして使いやすく可読性も高いため推奨されています。

次の例では `date` コマンドで現在の日付を取得し、その結果を使ってメッセージを表示しています。

```bash
echo "Today is $(date)"
```
